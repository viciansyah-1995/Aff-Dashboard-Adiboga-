# Mermaid Detail Flows

Folder ini disiapkan untuk menyimpan Mermaid flow detail per modul pada project **Affiliate Dashboard (Adiboga)**.

Tujuan folder ini:
- memecah flow global menjadi flow yang lebih detail
- memudahkan diskusi per modul
- menjadi dasar untuk PRD, wireframe, dan implementasi

## Rencana isi modul
1. `01-sourcing-flow.md`
2. `02-master-affiliator-flow.md`
3. `03-outreach-flow.md`
4. `04-campaign-and-portal-flow.md`
5. `05-sample-fulfillment-flow.md`
6. `06-content-submission-flow.md`
7. `07-point-and-reporting-flow.md`

## Catatan
- setiap file akan berisi diagram Mermaid + penjelasan singkat
- fokus diagram adalah alur operasional dan keputusan penting
- format dibuat agar mudah dipahami stakeholder dan tim produk
