# Aff Dashboard (Adiboga)

Workspace ini dipakai untuk perancangan sistem **Affiliate Dashboard** untuk Adiboga.

Scope besar project:
- sourcing affiliator
- listing dan enrichment data affiliator
- approach affiliator (AI + human)
- portal affiliator
- pemilihan SoW / campaign
- request & pengiriman sample via 3PL
- submission link konten
- point system
- reporting performa & estimasi ROI

Dokumen awal yang disiapkan:
- `BRD.md` — Business Requirements Document

Tahap saat ini:
- analisa flow bisnis
- identifikasi aktor, modul, data, dependency, dan risiko
- penyusunan BRD sebelum implementasi sistem
